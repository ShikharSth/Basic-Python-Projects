# 🚀 Basic Python Projects

A collection of beginner-friendly Python projects to practice core programming concepts like user input, conditionals, error handling, and basic operations.

---

## 📌 Project List

<details>
  <summary><strong> 🧮 1. Simple Calculator</strong></summary>
    A command-line calculator that performs basic arithmetic operations.

    #### ✨ Features
    - Supports operations: `+`, `-`, `*`, `/`, `%`
    - Handles invalid input using exception handling
    - Prevents crashes from incorrect user input

    ---

    ## ⚙️ Getting Started

    ### 📥 Clone the Repository
    ```bash
    git clone git@github.com:ShikharSth/Basic-Python-Projects.git
    ```

    ### 📂 Navigate to the Project Directory
    ```bash
    cd Basic-Python-Projects
    ```

    ### ▶️ Run the Calculator
    ```bash
    python simple_calculator.py
    ```

    ---

    ## 🧠 How It Works?

    1. Takes user input for two numbers  
    2. Converts input into numeric values  
    3. Performs the selected operation  
    4. Uses `try-except` to handle invalid input safely  

    ---

    ## ⚠️ Requirements

    Make sure Python is installed on your system.

    Download Python from:  
    https://www.python.org

    ---

    ## 📸 Sample Output

    ```bash
    Enter the first number: 10
    Enter the second number: 5
    Choose an operation (+, -, *, /, %): *

    The result of 10.0 * 5.0 is: 50.0
    ```

    ---

    ## 🛠️ Tech Stack

    - Python 3
